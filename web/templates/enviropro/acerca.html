Acerca del proyecto AgroDataLab EnviroPro

AgroDataLab EnviroPro es la práctica intermodular del programa educativo Horizonte Verde Digital, que conecta el sector agrario con la familia profesional de Informática y Comunicaciones.

Objetivos

  • Transformar datos reales de sensores agrícolas en información útil.
  • Analizar humedad, temperatura, batería y panel solar del nodo EnviroPro.
  • Detectar situaciones de riesgo relativo y generar alertas con recomendaciones prudentes.
  • Entrenar un modelo predictivo sencillo que anticipe periodos de sequedad.
  • Visualizar los resultados mediante una aplicación web Django desplegada en hosting.

Alcance

El sistema no emite órdenes agronómicas absolutas. No conoce el tipo de cultivo, el riego aplicado, la profundidad exacta de cada sensor ni la lluvia local. Las recomendaciones se formulan en términos de riesgo relativo y deben ser contrastadas con el equipo agrario.

Stack técnico

  • Análisis: Python 3.10, Pandas, NumPy, Matplotlib, Scikit-learn.
  • Web: Django 5 con Bootstrap 5 y WhiteNoise para estáticos.
  • Base de datos: SQLite (desarrollo), PostgreSQL (producción opcional).
  • Despliegue: Render / PythonAnywhere / Railway (cualquier hosting Django).

Equipo

Proyecto realizado por Diego dentro del programa educativo Horizonte Verde Digital.

Limitaciones y mejoras futuras

  • Integración con la API REST de IFAPA-RIA (ampliación opcional documentada en el notebook).
  • Mejora del modelo predictivo con técnicas no lineales.
  • Notificación push de alertas críticas (Telegram, email).
